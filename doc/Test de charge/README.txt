Problèmes en lien avec les tests de charges sur les VM Microsoft Azure du Cégep
- Connection se coupe à des moment aléatoire
- Vlan non configurable (aléatoire à la création puisqu'on a pas les droits d'admin)
- Les machines qui accepte d'être créer se corrompent lors de l'installation d'un OS ou de la modification matériel
- Pas les droits de supprimer une machine en cas de besoin
- Ajout d'un OS sur le réseau fastidieux et lent à apparaitre sur l'outil de virtualisation
- Plantage de l'outil lors de passage de droit sur la machine
- Pas de documentation et difficulté d'avoir des informations sur le fonctionnement des VMs du Cégep
